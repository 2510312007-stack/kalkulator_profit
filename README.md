# Kalkulator Profit

Aplikasi web berbasis Node.js untuk menghitung profit penjualan.

## Fitur
- Input harga beli
- Input harga jual
- Input jumlah
- Perhitungan profit otomatis

## Teknologi
- Node.js
- Express.js
- EJS
- Sequelize
- SQLite

## Cara Menjalankan
1. Clone repository
2. Jalankan `npm install`
3. Jalankan `node app.js`
4. Buka browser: http://localhost:3000
